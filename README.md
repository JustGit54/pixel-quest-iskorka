# Pixel Quest: Искорка

Кроссплатформенная мобильная пиксельная игра для Android, iPhone и iPad. Игровая часть написана на Phaser, нативные оболочки создаёт Capacitor.

## Что реализовано

- сюжет с тремя мирами: Лес Люмии, Пещеры Эха и Небесный Замок;
- платформенный геймплей: движение, прыжок, монеты, враги, осколки Света и финальный босс;
- крупное сенсорное управление, пауза, виброотклик и сохранение прогресса на устройстве;
- горизонтальная ориентация, адаптивный интерфейс и собственные пиксельные ассеты;
- исходники нативных проектов Android и iOS создаются командами Capacitor.

## Быстрый старт

Требуется Node.js 22+.

```bash
npm install
npm run build
npx cap sync
```

Для запуска веб-версии:

```bash
npm run dev
```

## Открытие нативных проектов

```bash
npm run cap:android  # Android Studio
npm run cap:ios      # Xcode (только macOS)
```

После каждого изменения игры запускай `npm run cap:sync`, затем собирай релиз внутри Android Studio или Xcode.

Папки `android/` и `ios/` уже включены в проект. Команду `npm run native:add` используй только для создания новой чистой копии платформы.
